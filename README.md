# 🎲 Verdades o Preguntas Incomodas - Juego para Adultos

Un juego interactivo de tarjetas diseñado para adultos, perfecto para reuniones con amigos donde se busca romper el hielo y pasar un momento divertido.

## 🎯 Características

- **Multiplayer**: Soporta de 2 a 10 jugadores
- **4 Categorías de Tarjetas**:
  - 🔴 **Verdad**: Preguntas íntimas y personales
  - 🟡 **Atrevido**: Retos y acciones audaces
  - 🟢 **Nunca He**: Confesiones y experiencias
  - 🔵 **¿Lo Harías?**: Escenarios hipotéticos
- **100+ Tarjetas**: Contenido variado y entretenido
- **Interfaz Moderna**: Diseño atractivo con animaciones
- **Turnos Automáticos**: Sistema de rotación de jugadores
- **Sin Repeticiones**: Evita que salgan las mismas tarjetas

## 🚀 Cómo Empezar

1. **Abrir el juego**: Abre el archivo `index.html` en tu navegador
2. **Agregar jugadores**: Ingresa los nombres de los participantes (mínimo 2)
3. **Empezar el juego**: Haz clic en "Empezar Juego"
4. **Jugar**: 
   - Cada jugador tendrá su turno
   - Haz clic en la tarjeta para revelar la pregunta
   - Responde o completa el reto
   - Pasa al siguiente turno

## 🎮 Cómo Jugar

### Reglas Básicas
- Los jugadores deben responder honestamente a las preguntas de "Verdad"
- En "Atrevido", los jugadores deben completar el reto propuesto
- En "Nunca He", quienes han hecho lo mencionado deben beber o realizar una penitencia
- En "¿Lo Harías?", responde si harías o no la situación planteada

### Configuración
- **Mínimo de jugadores**: 2
- **Máximo de jugadores**: 10
- **Duración**: Ilimitada, depende del grupo
- **Edad recomendada**: +18 años

## 🔧 Personalización

### Agregar Nuevas Tarjetas
Para agregar más tarjetas, edita el archivo `script.js` en la sección `cards`:

```javascript
const cards = {
    truth: [
        "Tu nueva pregunta de verdad aquí",
        // ... más preguntas
    ],
    dare: [
        "Tu nuevo reto aquí",
        // ... más retos
    ]
    // ... otras categorías
};
```

### Modificar Colores
Puedes personalizar los colores de los avatares modificando el array `avatarColors`:

```javascript
const avatarColors = [
    '#FF6B6B', '#4ECDC4', '#45B7D1',
    // ... tus colores personalizados
];
```
